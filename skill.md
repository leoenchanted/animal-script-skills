# 动物科普文案生成 Skill（优化版）

## 触发条件

当用户输入类似以下内容时自动触发：
- "我想了解[动物名]"
- "给我介绍一下[动物名]"
- "帮我写一篇关于[动物名]的科普文案"

---

## 执行流程

### 第一阶段：生成文字文案

#### Step 1：检查模板

读取 `.claude/templates/animal-template.md`：
- **有内容** → 按模板结构生成纯文字文案
- **无内容** → 询问用户提供模板

> 注意：模板只需关注**文字结构**，不需要图片占位符。图片和参考文献由 Skills 自动处理。

#### Step 2：深度多维度搜索（12维度）

| 维度 | 搜索关键词 | 目标信息 |
|:---|:---|:---|
| 基本特征 | `{animal} size weight height measurements` | 详细体型数据 |
| 外形细节 | `{animal} appearance fur color pattern features` | 毛色、纹路、构造 |
| 分类演化 | `{animal} taxonomy evolution scientific name` | 学名、分类、演化 |
| 生活习性 | `{animal} behavior habits social lifestyle` | 社交、活动规律 |
| 地理分布 | `{animal} geographic range distribution habitat` | 详细分布区域 |
| 饮食细节 | `{animal} diet prey food hunting feeding` | 食物、捕猎方式 |
| 繁殖完整 | `{animal} reproduction breeding mating lifecycle` | 求偶、育幼过程 |
| 生命周期 | `{animal} lifespan growth development stages` | 各阶段特点 |
| 保护现状 | `{animal} IUCN conservation status endangered` | 等级、威胁、保护 |
| 生态角色 | `{animal} ecological role ecosystem impact` | 生态系统作用 |
| 科学发现 | `{animal} research discoveries scientific facts` | 最新研究 |
| 文化趣闻 | `{animal} interesting facts cultural significance` | 冷知识、文化意义 |

#### Step 3：按模板生成纯文字文案

生成 **1500-3000字** 的 Markdown 文案，包含：
- 所有12维度的详细信息
- 具体数据和科学事实
- **不包含图片**（图片在第二阶段处理）

---

### 第二阶段：附加图片和参考文献

#### Step 4：搜索并验证图片

搜索以下类型图片：
- 主体照片、外形特写、栖息环境、行为瞬间
- 幼崽照片、群居场景、分布地图、对比图片

**图片验证流程**：
1. 获取图片URL列表
2. 使用 WebFetch 验证每个URL是否可访问
3. 只保留响应状态正常的图片
4. 过滤掉无效、过期、需登录的链接
5. 最终保留 **3-6张有效图片**

#### Step 5：整理参考文献

收集所有搜索到的权威来源：
- National Geographic
- IUCN Red List
- WWF
- Wikipedia/Britannica
- 学术论文/科研报告

- 参考链接 → 新页面打开来源页面

## 模板编写说明

### 简化模板（只需关注文字）

用户模板只需指定**文字结构**，例如：

```markdown
# [动物名称]科普解说

## 开篇介绍
用200字介绍这种动物，突出亮点。

## 外形特征
描述外形特点，包含具体数据。

## 生活习性
描述生活习性、饮食、社交等。

## 繁殖方式
描述繁殖过程、育幼方式。

## 保护现状
说明濒危等级和威胁。

## 趣味知识
列举3-5条冷知识。
```

### 图片和参考文献

**自动处理**，不需要在模板中指定：
- 图片会自动搜索、验证、附加到HTML底部
- 参考文献会自动整理、附加到HTML底部
- 所有链接可点击，在新页面打开

  - 右侧：实时预览 + 图片画廊 + 参考来源
