# 动物科普文案生成 Skill（图文版）

## 触发条件

当用户输入类似以下内容时自动触发：
- "我想了解[动物名]"
- "给我介绍一下[动物名]"
- "帮我写一篇关于[动物名]的科普文案"

---

## 执行流程

### 第一步：检查模板

读取模板文件，有内容则使用，无内容则询问用户。

### 第二步：深度多维度搜索

#### 文字信息（12维度）

| 维度 | 目标信息 |
|:---|:---|
| 基本特征 | 详细体型数据 |
| 外形细节 | 毛色、纹路、构造 |
| 分类演化 | 学名、演化历史 |
| 生活习性 | 社交、活动规律 |
| 地理分布 | 详细分布区域 |
| 饮食细节 | 具体食物、捕猎方式 |
| 繁殖完整 | 求偶、育幼全过程 |
| 生命周期 | 各阶段特点 |
| 保护现状 | IUCN等级、威胁 |
| 生态角色 | 生态系统作用 |
| 科学发现 | 最新研究 |
| 文化趣闻 | 文化意义、冷知识 |

#### 图片资源（8类型）

主体照片、外形特写、栖息环境、行为瞬间、幼崽照片、群居场景、分布地图、对比图片

---

### 第三步：整合生成图文文案

每张图片标注来源：
```
![描述](URL)
*图片来源：[来源名](来源URL)*
```

---

### 第四步：输出HTML编辑器

左侧Markdown源码（可编辑），右侧实时预览（图片显示）

---

## 模板编写指南

图片占位符：`![主体照片]` `![外形特写]` `![行为照片]` `![幼崽照片]` 等

详见 template-guide.md
