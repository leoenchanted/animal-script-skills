# 动物科普短视频脚本 Skill

## 触发与输出

当用户输入包含“想了解”“介绍一下”“写一篇关于”“科普文案”“视频脚本”“分镜”“生图提示词”等表达，且主题涉及动物或生物时，使用本 skill。

默认输出：Markdown 视频脚本、分镜表、中英文 AI 生图提示词、完整口播稿、剪辑建议、事实参考来源。用户需要页面时，额外输出可编辑 HTML 分镜工作台。

## 核心目标

生成适合短视频制作的动物科普脚本，而不是普通文章。输出必须帮助用户直接进入视频制作流程：

1. 先搜索权威资料，保证动物事实准确。
2. 再生成 60-90 秒短视频解说脚本。
3. 将脚本拆成 10-15 个镜头。
4. 每个镜头提供对应画面描述、中文 AI 生图提示词、英文 AI 生图提示词和动效建议。
5. 默认视觉方向为卡通科普风，不直接提供网络图片链接，除非用户明确要求真实图片素材。

## 执行流程

### 第一步：确认输入

从用户请求中提取：

- 动物名称
- 视频平台或用途（如抖音、视频号、B站、小红书）
- 期望时长（未指定时默认 60-90 秒）
- 视觉风格（未指定时默认卡通科普风）
- 目标受众（未指定时默认大众科普）

如果用户只给出动物名称，也直接执行，不要反复追问。

### 第二步：检查模板

优先读取项目模板：

`.claude/templates/animal-video-template.md`

如果该文件存在且有内容，按用户模板生成。否则使用本 skill 的默认视频脚本结构。

### 第三步：权威资料搜索

围绕动物主题进行多维度搜索。事实必须来自可核验来源，不要编造数据。

优先来源：

1. IUCN Red List
2. WWF
3. National Geographic
4. Britannica
5. Smithsonian / San Diego Zoo / AZA 等动物机构
6. 学术机构、博物馆、政府保护组织

搜索维度：

| 维度 | 目标信息 |
|:---|:---|
| 基本档案 | 学名、分类、体型、体重、寿命 |
| 外形特征 | 毛色、纹路、身体构造、辨识点 |
| 栖息地 | 地理分布、典型环境、气候条件 |
| 生活习性 | 活动时间、社交方式、领地行为 |
| 饮食策略 | 食物类型、捕食方式、取食技巧 |
| 繁殖育幼 | 求偶、产仔/产卵、育幼方式 |
| 生存技能 | 特殊能力、防御方式、适应机制 |
| 保护现状 | IUCN 等级、种群趋势、主要威胁 |
| 反差亮点 | 适合短视频开场的惊讶点 |
| 冷知识 | 适合中段节奏变化的趣味事实 |

### 第四步：生成视频策划

输出脚本前先给出视频基础设定：

- 视频标题
- 一句话爆点
- 视频时长
- 目标平台
- 叙事节奏
- 统一视觉风格
- 主角角色设定

统一角色设定必须让每张 AI 图保持一致，包括：

- 动物外形关键词
- 卡通化程度
- 色彩氛围
- 光影质感
- 画幅比例
- 负面限制（无文字、无水印、不要真实照片风等）

### 第五步：生成分镜表

默认输出 10-15 个镜头。每个镜头必须包含：

| 字段 | 要求 |
|:---|:---|
| 镜头 | 两位数编号，如 01 |
| 时间 | 例如 0-5s |
| 解说词 | 可直接配音的中文口播 |
| 画面内容 | 视频中应出现的画面 |
| 中文生图提示词 | 可复制到国产 AI 生图工具 |
| English Prompt | 可复制到 Midjourney / DALL-E / SD |
| 动效建议 | 推拉摇移、字幕节奏、转场、局部动画 |
| 事实依据 | 简短标注来自哪个来源或哪类资料 |

提示词要求：

- 使用卡通科普、动画短片、儿童科普绘本、3D 卡通或干净插画等风格词。
- 每条提示词都要包含主体、场景、动作、情绪、构图、光影、比例。
- 默认比例为 16:9。
- 默认加入限制：无文字、无水印、无 logo、不要真实照片风。
- 不要要求生成真实摄影作品，除非用户明确指定。
- 不要让提示词包含虚假的文字标签或难以稳定生成的小字。

### 第六步：输出完整口播稿

在分镜表后输出一版连续口播稿，方便用户直接录音。

要求：

- 语言自然，有短视频节奏。
- 数据表达清楚但不过度堆砌。
- 重要事实与分镜表一致。
- 结尾有评论互动或情绪收束。

### 第七步：输出制作清单

最后输出：

- AI 生图清单
- 动效剪辑建议
- 配音语气建议
- 背景音乐建议
- 事实参考来源

事实来源和 AI 画面提示词必须分开，不要把 AI 图当作真实图片来源。

## 默认输出结构

```markdown
# [动物名称]科普短视频脚本

## 视频设定

## 统一角色设定

## 分镜脚本

| 镜头 | 时间 | 解说词 | 画面内容 | 中文生图提示词 | English Prompt | 动效建议 | 事实依据 |
|:---|:---|:---|:---|:---|:---|:---|:---|

## 完整口播稿

## AI 生图清单

## 剪辑与配音建议

## 事实参考来源
```

## HTML 展示要求

如用户需要 HTML 页面，输出一个可编辑分镜工作台：

- 顶部展示标题、动物、时长、风格。
- 主区域展示分镜卡片，而不是普通文章。
- 每张卡片展示镜头编号、时间、解说词、画面内容、中文提示词、英文提示词、动效建议。
- 提供复制提示词按钮。
- 提供完整口播稿区域。
- 风格应像视频制作工作台，适合用户逐镜头复制到 AI 生图工具。

## 质量标准

- 所有动物事实必须可追溯到来源。
- 不要为了戏剧效果编造体型、速度、寿命、分布或保护等级。
- 不要把多个相近物种的数据混在一起，除非明确说明是在讲整个类群。
- 不要输出真实图片 URL 作为主要素材；本 skill 的主要交付物是 AI 生图提示词。
- 画面提示词要服务于视频节奏，每段解说都应有对应画面。
