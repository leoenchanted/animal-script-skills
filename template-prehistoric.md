# 史前生物穿越现代沉浸式解说模板

用途：生成 60-90 秒「史前生物穿越现代」沉浸式解说短视频脚本。采用 10 步情绪递进结构：
**压迫 → 爽 → 更大压迫 → 登神 → 文明碾压**，以电影级分镜标准输出。

---

## 视频设定

- **史前生物名称**：[生物名称]
- **视频标题**：[强反差标题，如"如果把XX扔进现代世界"]
- **一句话爆点**：[核心悬念，如"它还能成为霸主吗"]
- **目标时长**：60-90 秒
- **目标平台**：抖音 / B站 / 视频号
- **叙事模型**：史前生物穿越现代沉浸式解说
- **情绪曲线**：压迫 → 爽 → 更大压迫 → 登神 → 文明碾压
- **视觉风格**：卡通科普风，画面明亮统一，适合 16:9 视频
- **叙事节奏**：跨时代假设 → 出生即巅峰 → 环境打压 → 猎杀爽点 → 环境背叛 → 成长压力 → 登神统治 → 人类文明出现 → 降维打击 → 哲学升华

**关键视觉标识**：[列出 3-4 个让这个生物在每张图中保持一致的视觉特征，如牙齿形状、翼膜纹理、鳍肢比例等]
