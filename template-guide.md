# 动物科普文案模板编写指南

本文件既是**默认模板**，也是**编写指南**。你可以直接使用，或参考此格式自定义。

---

## 📝 如何编写带图片的模板？

### 一、基本概念

模板是你告诉 Claude "文案要写成什么样" 的参考格式。你只需要：

1. **指定结构**：告诉 Claude 你想要的章节划分
2. **指定图片位置**：用占位符标记哪里需要图片
3. **指定内容类型**：说明每个部分写什么

Claude 会根据你的模板，自动搜索资料和图片填充内容。

---

### 二、图片占位符使用

在模板中，用以下占位符告诉 Claude 在哪里放图片：

| 占位符 | 用途 | 推荐位置 |
|:---|:---|:---|
| `![主体照片]` | 动物的主要形象照片 | 开篇/标题下方 |
| `![外形特写]` | 展示局部特征的照片 | 外形特征章节 |
| `![行为照片]` | 捕捉行为瞬间的照片 | 生活习性章节 |
| `![幼崽照片]` | 幼年状态的照片 | 繁殖章节 |
| `![栖息环境]` | 生活环境照片 | 栖息地章节 |
| `![分布地图]` | 地理分布图 | 分布章节 |
| `![群居场景]` | 群体生活照片 | 社会结构章节 |
| `![对比图片]` | 种类对比图 | 基本档案章节 |

**重要**：每张图片后面加来源标注：
```markdown
![图片描述](图片URL)
*图片来源：[来源名称](来源页面URL)*
```

### 三、模板示例

#### 示例 A：简洁图文版（推荐新手）

```markdown
# [动物名称]科普解说

![主体照片]

## 开篇介绍
用200字介绍这种动物，突出一个吸引人的亮点。

## 外形特征
![外形特写]
描述这种动物的外形特点，包含具体数据（体长、体重等）。

## 生活习性
![行为照片]
描述这种动物怎么生活、吃什么、有什么特殊技能。

## 繁殖方式
![幼崽照片]
描述这种动物怎么繁殖、育幼方式有什么特别。

## 保护现状
说明这种动物的濒危等级和主要威胁。

## 参考来源
列出所有文字和图片来源。
```

### 四、写给别人的模板说明

如果你把这个 skills 分享给别人，告诉他们：

1. **创建模板文件**：
   在项目 `.claude/templates/animal-template.md` 中放入你的模板

2. **模板可以简单也可以复杂**：
   - 简单版：只指定章节和图片位置
   - 复杂版：指定字数、数据要求、对比表格等

3. **图片会自动搜索**：
   Claude 会根据占位符自动搜索对应的图片，并标注来源

4. **如果没有模板**：
   Claude 会询问你提供模板格式
