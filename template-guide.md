# 动物科普文案模板编写指南（简化版）

## 📝 核心概念

**模板只需关注文字结构，图片和参考文献由 Skills 自动处理。**

---

## 一、模板是什么？

模板是你告诉 Claude "文案要写成什么样" 的参考格式。

你只需要：
1. **指定章节结构**：告诉 Claude 你想要的章节划分
2. **指定内容类型**：说明每个部分写什么
3. **不需要图片占位符**：图片会自动搜索并附加到HTML底部

---

## 二、简单模板示例

### 示例 A：基础版（推荐新手）

```markdown
# [动物名称]科普解说

## 开篇介绍
用200字介绍这种动物，突出一个吸引人的亮点。

## 外形特征
描述外形特点，包含具体数据（体长、体重等）。

## 生活习性
描述怎么生活、吃什么、有什么特殊技能。

## 繁殖方式
描述繁殖过程、育幼方式有什么特别。

## 保护现状
说明濒危等级和主要威胁。

## 趣味知识
列举3-5条有趣的冷知识。
```

## 三、图片和参考文献怎么处理？

**完全自动，你不需要操心：**

### 图片处理流程

1. Claude 搜索8类型图片（主体、特写、行为、幼崽等）
2. **自动验证每个图片URL是否有效**
3. 只保留能正常打开的图片
4. 附加到HTML底部的"图片画廊"区域
5. 点击图片在新页面打开原图

### 参考文献处理流程

1. Claude 收集所有搜索到的权威来源
2. 整理成"文字来源"和"图片来源"两部分
3. 附加到HTML底部的"参考来源"区域
4. 点击链接在新页面打开原文

## 四、使用说明

### 步骤 1：创建模板文件

在项目目录创建：
```
.claude/templates/animal-template.md
```

### 步骤 2：粘贴你的模板

把上面任意一个示例粘贴进去，或自己写一个结构。

### 步骤 3：使用

告诉 Claude：
```
我想了解企鹅
```

Claude 会：
1. 读取你的模板
2. 搜索12维度信息
3. 按模板生成纯文字文案
4. 自动搜索并验证图片
5. 输出HTML（文案+图片画廊+参考文献）

## 五、常见问题

### Q：模板里需要写图片位置吗？

**不需要**。图片会自动搜索并放在HTML底部。

### Q：图片链接会失效吗？

**不会**。Skills 会验证每个图片URL，只保留有效链接。

### Q：参考文献从哪里来？

自动从搜索来源整理，包括 National Geographic、IUCN、WWF 等权威机构。

### Q：点击链接会跳转到哪里？

所有链接在**新页面**打开，不影响当前编辑。
