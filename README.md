# Animal Video Script — 动物科普短视频脚本生成器

一个 Claude Code Skill，自动生成**可直接用于视频制作**的短视频脚本 + 交互式 HTML 分镜工作台。支持两种叙事模型，覆盖常规动物科普和「史前生物穿越现代」沉浸式解说。

---

## 它能做什么

对 Claude 说一句话，它就能：

1. **联网搜索权威资料**（IUCN、WWF、National Geographic 等）
2. **自动选择叙事模型** — 判断你想要标准科普还是史前穿越爽文
3. **生成完整分镜脚本** — 逐镜头的解说词、画面、AI 生图提示词
4. **输出可编辑 HTML 分镜工作台** — 浏览器直接打开，支持编辑和一键复制
5. **静态验证** — 自动检查内容完整性，防止遗漏

---

## 两种叙事模型

| 维度 | 标准科普模型 | 史前穿越沉浸式解说模型 |
|------|-------------|---------------------|
| **适用场景** | 现存动物、一般科普需求 | 已灭绝动物、穿越/复活设定 |
| **典型指令** | "帮我写一篇关于雪豹的科普脚本" | "如果把风神翼龙扔进现代非洲，它能活吗" |
| **镜头数** | 10-15 个 | **固定 10 个**，对应完整情绪曲线 |
| **叙事结构** | 开场反差→外形→栖息地→饮食→技能→习性→繁殖→冷知识→保护→收尾 | **跨时代假设→出生即巅峰→环境打压→第一次猎杀→环境背叛→成长压力→登神巅峰→人类文明出现→降维打击→哲学升华** |
| **情绪曲线** | 认知驱动，平稳递进 | **压迫→爽→更大压迫→登神→文明碾压** |
| **分镜字段** | 解说/画面/提示词/动效/来源 | 以上全部 + **9 个电影级字段** |
| **搜索策略** | 单线搜索（现代动物资料） | **双线搜索**（古生物学数据 + 现代环境参照数据） |

### 史前穿越模型的 10 步情绪设计

```
跨时代假设  ──▶  出生即巅峰  ──▶  环境打压  ──▶  第一次猎杀  ──▶  环境背叛
  (神秘)         (震撼)         (危机)         (爽感)         (孤独)
     │                                                          │
     ▼                                                          ▼
  哲学升华  ◀──  降维打击  ◀──  人类文明出现  ◀──  登神巅峰  ◀──  成长压力
  (余味)         (悲剧)         (压迫)         (崇高)         (焦灼)
```

### 每个镜头的 9 个电影级字段

在史前穿越模型下，每个镜头输出完整的电影分镜设定：

| 字段 | 说明 | 示例 |
|------|------|------|
| **景别** | 取景范围 | 大远景、远景、全景、中景、近景、特写、大特写 |
| **主体** | 画面核心对象 | 利维坦鲸幼崽、风神翼龙完全体、核潜艇 |
| **动作** | 主体在做什么 | 缓慢游动、俯冲猎杀、挣扎坠落 |
| **环境** | 场景地点与氛围 | 太平洋深海、非洲草原黄昏、工业港口 |
| **光线** | 照明与色调 | 深海蓝光、黄昏逆光、冷峻工业光 |
| **运镜** | 摄影机运动方式 | 缓慢推进、环绕仰拍、手持晃动、慢动作定格 |
| **情绪** | 镜头传递的情感 | 神秘压迫、爽感震撼、孤独悲凉、悲剧无力 |
| **风格** | 全片统一的视觉风格 | 卡通科普动画 |
| **设定** | 世界观前提 | 1,300万年前灭绝的巨兽出现在现代太平洋 |

## 使用方法

### 标准科普

直接告诉 Claude 动物名称：

```
帮我写一篇关于雪豹的科普短视频脚本，要卡通生图提示词
我想了解蓝鲸，做成 60 秒分镜脚本
介绍一下大王乌贼，需要完整口播稿
```

### 史前穿越沉浸式解说

触发关键词（穿越、能不能存活、扔进现代等），Claude 会自动识别：

```
如果把利维坦鲸扔进现代太平洋，它还能成为霸主吗
风神翼龙穿越到现代非洲，它能活下来吗
如果把棘龙放进现在的刚果河，会发生什么
```

你也可以手动指定叙事风格：

```
我想做一个史前动物穿越到现代的沉浸式解说
用风神翼龙那种叙事风格，写一只巨齿鲨穿越到现代
```

## 交付物

每次请求，Claude 会同时在对话中输出 Markdown 脚本，并在本地创建 HTML 文件。

### 标准科普模式输出

```markdown
# [动物名称]科普短视频脚本

## 视频设定
## 统一角色设定
## 分镜脚本
| 镜头 | 时间 | 解说词 | 画面内容 | 中文提示词 | English Prompt | 动效建议 | 事实依据 |
## 完整口播稿
## AI 生图清单
## 剪辑与配音建议
## 事实参考来源
```

同时生成：`[animal-name]-storyboard.html`（可编辑分镜工作台）

### 史前穿越模式输出

```markdown
# [史前生物]穿越现代沉浸式解说脚本

## 视频设定（含叙事模型、情绪曲线）
## 统一角色设定
## 数据对照表（史前 vs 现代环境参数对比）
## 分镜总表
| 镜头 | 时间 | 叙事功能 | 解说词 | 画面内容 | 景别 | 运镜 | 情绪 | 动效建议 | 事实依据 |

## 镜头详细设定（每个镜头独立展示）
### 镜头 01 — 跨时代假设 (0-8s)
| 景别 | 主体 | 动作 | 环境 | 光线 | 运镜 | 情绪 | 风格 | 设定 |
（完整 9 字段表格 + 解说词 + 画面内容 + 中英文提示词 + 动效 + 来源）

## 完整口播稿
## AI 生图清单
## 剪辑与配音建议（含情绪节奏音乐方向）
## 事实参考来源
### 古生物学数据
### 现代环境参照数据
```

同时生成：`[prehistoric-animal]-storyboard.html`（含电影分镜设定编辑区）

## 安装

### 项目级安装

复制整个文件夹到项目的 `.claude/skills/` 目录：

```text
your-project/.claude/skills/animal-video-script/
├── SKILL.md
├── template-default.md
├── template-prehistoric.md
├── template-guide.md
├── template-storyboard.html
└── examples/
    └── style-templates/
        ├── prehistoric-leviathan-survival.txt
        └── prehistoric-pterosaur-survival.txt
```

### 全局安装

复制到 `~/.claude/skills/animal-video-script/`。

---

## 自定义模板

在项目中创建 `.claude/templates/animal-video-template.md`，Claude 会优先读取你的自定义模板。格式参考 `template-guide.md`。
